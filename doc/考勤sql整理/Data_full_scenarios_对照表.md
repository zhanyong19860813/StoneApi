| 班次 | 班次说明 | 是否跨天 |

| BC_4CARDS, BC_4CARDS_M100, BC_4CARDS_001 | 四次卡 | 不跨天 / 上班跨天 / 下班跨天 |

- **上班跨天**：班次上班在前一日（如 BC_2CARDS_M10、BC_4CARDS_M100）
- **下班跨天**：班次下班在次日（如 BC_2CARDS_P01、BC_4CARDS_001）
- **不跨天**：上下班均在当日（如 BC_2CARDS、BC_4CARDS、BC_6CARDS）

## 使用说明

1. **Excel/CSV 对照表**：`Data_full_scenarios_对照表.csv` 含【班次说明】【是否跨天】两列，可直接用 Excel 打开对照检查
2. 执行 `Data_full_scenarios_202603.sql` 生成日结果
3. 用本表逐行对照 `att_lst_DayResult` 或视图 `v_att_lst_DayResult`
4. 重点关注：attStatus、attStatus1/2/3、attLate、attEarly、attAbsenteeism 与预期是否一致
