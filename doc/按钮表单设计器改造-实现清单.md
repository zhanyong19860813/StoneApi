## 三、调用流程（改造后）

```
点击工具栏按钮（如 add / edit）
  → handleToolbarClick(btn)
  → 有 actionModule 且存在 handler？
       → 是：执行 handler，按返回值处理（openModal / openFormTabsTableModal / openFormFromDesigner）
       → 否：btn.form_code 存在？
            → 是：requiresSelection ? 取 selectedRows[0] : {} → openFormFromDesignerModal(formCode, initialValues)
            → 否：提示未配置
  → 表单提交成功 → onSuccess → gridApi.reload()
```
